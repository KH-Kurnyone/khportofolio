Pengalaman

Berikut adalah deskripsi pengalaman saya selama merintis karir di bidang informatika

2020 - 2023 adalah langkah awal saya dibidang Rekayasa Perangkat Lunak di SMK Negeri 3 Banjar, saya berlajar tentang dasar-dasar pemrograman web mulai dari database, desain layout, dan develop atau coding web. Semasa SMK juga saya aktif di organisasi menjadi sekretaris, disana saya belajar banyak seputar softskill, manajemen, dan skill administrasi.

2023 - Saat ini saya melanjutkan karir saya ke jenjang perguruan tinggi di Politeknik LP3I Kampus Tasikmalaya pada prodi Manajemen Informatika. Disini saya ingin memperdalami lebih lanjut apa yang telah saya dapatkan saat SMK. Saya juga aktif berorganisasi, pernah mengikuti beberapa lomba IT, dan mengerjakan beberapa freelance.

2022 saat memasuki kelas 11 SMK selama 3 bulan saya melaksanakan PKL di software house Access Media Tasikmalaya. Disana saya belajar bagaimana peluang bisnis dari bidang RPL ini dan juga saya turut membantu dalam pembuatan video tutorial penggunaan dari website atau aplikasi yang telah dibuat.

2024 memasuki perkuliahan saya kembali magang untuk mengisi waktu libur selama 2 bulan, saat itu saya belajar mengembangkan sebuah webiste yang sudah ada.

Saya juga aktif kembali berorganisasi dibidang komputer yaitu UKM LCC sebagai Ketua, disini saya belajar lanjutan pengembangan dari softskill dan manajemen saya serta hardsklill saya salah satunya membuat sebuah webiste pencatatan kehadiran untuk setiap kegiatan UKM LCC.
